# Tech Stack: Life Habits

## Frontend
- **Framework**: React 19 (Using Hooks and Context API for state management)
- **Language**: TypeScript (Ensuring type safety across the application)
- **Styling**: Tailwind CSS v4 (Utilizing the latest PostCSS-based engine for high-performance styling)
- **Icons**: Lucide React (Clean, lightweight stroke-based icons)
- **Charts**: Recharts (Composability-first charting library for time/money visualization)
- **Markdown**: React Markdown (For rendering AI-generated insights)

## Data & Storage
- **Database**: Dexie.js (A wrapper for IndexedDB, providing a robust local-first storage solution)
- **Sync Protocol**: WebDAV (Standardized protocol for private cloud synchronization like Nutstore or Nextcloud)

## Intelligence
- **AI Engine**: Google Gemini API (Using `@google/genai` SDK)
- **Primary Model**: `gemini-2.0-flash` (Optimized for speed and insightful analysis)

## Tooling & Infrastructure
- **Build Tool**: Vite 6 (Fast ESM-based build system)
- **Package Manager**: npm
- **Environment**: Node.js 18+
- **Testing Framework**: Vitest (Vite-native unit testing framework)
- **Testing Library**: React Testing Library & jest-dom
- **Database Mocking**: fake-indexeddb
- **Coverage Tool**: v8
